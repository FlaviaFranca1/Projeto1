git init - Este comando é usado para criar um novo repositório GIT git add - pode ser usado para adicionar arquivos ao índice git clone - é usado para fins de verificação de repositório git commit - é usado para confirmar as alterações na head git status - exibe a lista de arquivos alterados juntamente com os arquivos que ainda não foram adicionados ou confirmados. git branch - pode ser usado para listar, criar ou excluir ramos.
